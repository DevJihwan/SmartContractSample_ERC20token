# SmartContractSample_ERC20Token
```
ERC20Toekn 발행 및 테스트넷 배포
ERC20이란 승인받은 개선 제안서 : EIP(Ethereum Imporvement Proposal) -> ERC(Ethereum Request for Comments)
대부분의 토큰들은 ERC20을 기반으로 만들어짐. 
```
